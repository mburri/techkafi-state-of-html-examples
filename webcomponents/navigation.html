Web Component

Web Components is a suite of different technologies allowing you to create reusable custom elements — with their functionality encapsulated away from the rest of your code — and utilize them in your web apps.

See: https://developer.mozilla.org/en-US/docs/Web/API/Web_components

See: https://www.puzzle.ch/de/blog/articles/2023/11/01/web-components-part-2-browser-apis

The navigation of this example app is a Web Component!

class Navigation extends HTMLElement { constructor() { super(); this.attachShadow({mode: "open"}); const nav = document.createElement("nav"); nav.innerHTML = ` <a href="index.html">Home</a> <details open> <summary>Forms</summary> <div> <a href="forms/form-data-api.html">Form Data API</a> <a href="forms/input-capture.html">Html attribute: capture</a> <a href="forms/select-list.html">Selectlist</a> </div> </details> // more details omitted ` const style = document.createElement("style") style.textContent = ` nav { display: flex; flex-direction: column; @media screen and (min-width: 900px) { width: 100%; flex-direction: row; justify-content: space-around; } } // more styles omitted ` this.shadowRoot.append(nav, style); } }
